# Visited Countries Tracker

A web application to track and display countries you've visited on an interactive world map.

## Features

- Add countries you've visited via a text input form
- View your list of visited countries
- Interactive world map showing visited countries in green
- Remove countries from the list
- Data persists in your browser's local storage
- Click on countries on the map to see their names

## How to Use

1. Open `index.html` in your web browser
2. Enter a country name in the input field (use full country names like "United States", "France")
3. Click "Add Country" to add it to your list and highlight it on the map
4. Use the "Remove" button next to each country to remove it from both the list and map
5. Zoom and pan the map to explore

## Files

- `index.html`: The main HTML structure with map and form
- `styles.css`: CSS styling for the application layout
- `script.js`: JavaScript for functionality, local storage, and map interaction

## Technologies Used

- Leaflet.js for the interactive map
- OpenStreetMap tiles
- World countries GeoJSON data
